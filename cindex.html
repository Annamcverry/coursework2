How To Dynamically Create HTML From JS

HTML can be rendered Statically or Dynamically.

Static

This means that no matter what happens on the page, each time the browser renders this HTML page it will always display the same.

Dynamically

This means that what has rendered on the page might differ dependant on different scenarios. IE, in your case, when a user selects a different price or location.
